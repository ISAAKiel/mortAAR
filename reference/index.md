# Package index

## Basic functions

- [`prep.life.table()`](https://isaakiel.github.io/mortAAR/reference/prep.life.table.md)
  : Creates the input for the function life.table
- [`life.table()`](https://isaakiel.github.io/mortAAR/reference/life.table.md)
  : Calculates a life table
- [`plot`](https://isaakiel.github.io/mortAAR/reference/plot.md) : Plot
  a mortaar_life_table or a mortaar_life_table_list
- [`print`](https://isaakiel.github.io/mortAAR/reference/print.md) :
  Encode and print a mortaar_life_table or a mortaar_life_table_list

## Advanced functions

- [`lt.representativity()`](https://isaakiel.github.io/mortAAR/reference/lt.representativity.md)
  : Checks the representativity of the non-adult age groups in a mortAAR
  life table
- [`lt.correction()`](https://isaakiel.github.io/mortAAR/reference/lt.correction.md)
  : Calculates a corrected life table from a mortAAR life table
- [`lt.reproduction()`](https://isaakiel.github.io/mortAAR/reference/lt.reproduction.md)
  : Calculates reproduction indices from a mortAAR life table
- [`lt.sexrelation()`](https://isaakiel.github.io/mortAAR/reference/lt.sexrelation.md)
  : Calculate masculinity index and maternal mortality from a mortAAR
  life table list
- [`lt.population_size()`](https://isaakiel.github.io/mortAAR/reference/lt.population_size.md)
  : Calculates population size from a mortAAR life table or numeric
  values
- [`halley.band()`](https://isaakiel.github.io/mortAAR/reference/halley.band.md)
  : Halley band of the mortality profile of a skeletal population

## Helper functions

- [`lt.indices()`](https://isaakiel.github.io/mortAAR/reference/lt.indices.md)
  : Generates indices from a mortAAR life table for use in other
  functions
- [`pop.sim.gomp()`](https://isaakiel.github.io/mortAAR/reference/pop.sim.gomp.md)
  : Simulation of a population of adults with Gompertz distribution
- [`random.cat()`](https://isaakiel.github.io/mortAAR/reference/random.cat.apply.md)
  [`random.cat.apply()`](https://isaakiel.github.io/mortAAR/reference/random.cat.apply.md)
  : Generation of random age ranges
- [`as.mortaar_life_table_list()`](https://isaakiel.github.io/mortAAR/reference/mortaar_life_table.md)
  [`as.mortaar_life_table()`](https://isaakiel.github.io/mortAAR/reference/mortaar_life_table.md)
  : mortaar_life_table and mortaar_life_table_list
