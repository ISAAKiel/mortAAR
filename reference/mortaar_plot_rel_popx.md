# Plots the mortality rate rel_popx for a single life table.

Plots the mortality rate rel_popx for a single life table.

## Usage

``` r
mortaar_plot_rel_popx(x, lty = 1, ...)
```

## Arguments

- x:

  an object of the class mortaar_life_table.

- lty:

  line type, default to 1.

- ...:

  further arguments passed to the print function.
