# Plots a coordinate system for life expectancy ex for a single life table.

Plots a coordinate system for life expectancy ex for a single life
table.

## Usage

``` r
mortaar_plot_ex_frame(x, my_subsets = "", n, ...)
```

## Arguments

- x:

  an object of the class mortaar_life_table.

- my_subsets:

  a vector of categories from the sublist of mortaar_life_table.

- n:

  number of individuals.

- ...:

  further arguments passed to the print function.
